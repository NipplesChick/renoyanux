Wroterrl Analytics

2021

  • Python (Anacondas)
  • Deep Learning
  • Big Data
  • Machine Learning
  • Classification
  • Decision Tree

A Decision Tree classification project in this context aims to build a model that can predict which salary bracket an employee at PT Zuresha Hitoma Jakarta belongs to, based on various features about them. Target Variable (The Salary): This is the variable we want to predict. For classification, the numerical salary must be converted into discrete classes (e.g., salary $\le$ IDR 5 million, IDR 5-10 million, IDR $\ge$ 10 million). Features (Predictor Variables): These are the attributes that influence the salary. Common features include: Years of Experience: (e.g., 3 years, 10 years), Education Level: (e.g., Bachelor's, Master's, Diploma), Job Title/Department: (e.g., Engineer, HR Staff, Manager), Age, Performance Rating: (e.g., Excellent, Good, Average), Location in Jakarta: (if relevant).

A Decision Tree works by repeatedly splitting the data into subsets based on the features that provide the most information gain or purity regarding the target variable (salary class). Structure: It resembles a flowchart or a tree with three main components: Root Node: Represents the entire dataset (all 187 employees), Internal Nodes: Represent a test on an attribute (a feature), e.g., "Is Experience > 5 years?", Leaf Nodes: Represent the final classification result (the predicted salary class), e.g., 'High Salary.'

This analysis helps the company ensure fairness and consistency in their compensation structure, identifying the key attributes that drive salary decisions within the 187 employees surveyed.
